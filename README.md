## Technologies Used

### AVR

* ATmega16 / ATmega32
* Embedded C
* AVR-GCC
* Proteus Simulation

### ARM

* LPC2148 / ARM7
* Embedded C
* Keil uVision
* Proteus Simulation

### Verilog

* Verilog HDL
* Xilinx Vivado / ISE
* ModelSim
* GTKWave

* GPIO Programming
* Timers and Counters
* Interrupts
* ADC
* PWM
* UART

* I2C

### ARM Microcontrollers

* GPIO Control
* Timer Programming
* UART Communication
* ADC and PWM
* Interrupt Handling
* Real Time Clock (RTC)
* Embedded Peripheral Interfacing

### Verilog HDL

* Combinational Logic
* Sequential Logic
* Finite State Machines
* Counters and Registers
* Arithmetic Logic Units
* Memory Design
* Testbench Development
